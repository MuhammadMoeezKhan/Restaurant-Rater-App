# Restaurant-Rater-App


Android Studio application that allows a user to rate restaurants that they have visited and store the reviews on their deviceâ€™s **Internal Storage**. The applcaiton Reads and Writes data from/to the internally linked **CSV** file.

#### Each **Review** stores:
* The restaurant's name 
* If the Restaurant is in the user's favorite list 
* The date of the Review
* The time of the Review
* The type of meal the user ate (breakfast, lunch, or dinner)
* The rating using an Android seekbar

Use this applicaiton to rate the next restaurants you visit!
